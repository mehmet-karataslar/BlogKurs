# BlogKurs

React + Next.js (App Router) + TypeScript projesi.

## Cursor ayarları

- **Kurallar:** `.cursor/rules/*.mdc` — React, Next.js, TypeScript kod stilleri (dosya/glob’a göre otomatik).
- **Beceriler:** `.cursor/skills/*/SKILL.md` — Sayfa, bileşen, API route ve veri çekme iş akışları.
- **Özet:** `AGENTS.md` — Teknoloji, yapı ve kural/skill listesi.
- **Rehber:** `.cursor/NOTLAR-Skills-Rules-Subagents-Commands.md` — Skills, rules, subagents, commands kullanımı.
- **Alt ajanlar:** `.cursor/SUBAGENTS.md` — explore, shell, generalPurpose ne zaman kullanılır; `.cursor/rules/subagents-usage.mdc` — AI için seçim kuralı.
